# 三个可交互 Demo + 作品集 · 同一国内域名

目标：HR 在 **`https://www.chenmengqiu-portfolio.xyz`** 上既能看你的主页，也能**直接体验**三个项目（不是只看截图）。

**不需要再买域名。** 所有路径都在一个域名下：

| 内容 | 地址 |
|------|------|
| 作品集 | `/` |
| UniPass 营销 + 演示 | `/demos/unipass/marketing/` |
| 本地出行 Agent（可点可玩，mock） | `/demos/meituan-agent/?demo=1` |
| joblens 落地页 | `/demos/joblens/` |

---

## 为什么现在 Vercel 反代不够？

反代仍走**海外**服务器，国内 iframe 容易白屏。要让 HR **稳定可交互**，需要：

1. **ICP 备案**（域名在阿里云，约 1～3 周）
2. 整站（含 `demos/`）放到 **阿里云 OSS + CDN（中国大陆）**
3. 域名解析从 Vercel **改到** 阿里云 CDN

---

## 第一步：构建三个项目到 `demos/`（本机）

在作品集目录打开 PowerShell：

```powershell
cd "d:\Users\86133\Desktop\cmq个人网站"
.\scripts\build-demos.ps1
```

脚本会：

- 复制 `joblens/website` → `demos/joblens/`
- 构建美团 Agent（**mock 模式**，国内无需 LLM API）→ `demos/meituan-agent/`
- UniPass：若 Next 静态导出报错（`app/api/*`），暂保留 Vercel 反代；要纯国内需先改 API 路由或单独导出营销页

构建成功后，双击 **`推送GitHub.bat`**（可选：过渡期仍走 Vercel，有本地文件会**优先**于反代）。

## 第二步：阿里云备案

1. 阿里云控制台 → **备案**
2. 主体：个人；网站：作品集展示
3. 备案域名：`chenmengqiu-portfolio.xyz`
4. 等审核通过（期间可继续用 Vercel + 自定义域名）

---

## 第三步：OSS 托管整站

### 1. 创建 Bucket

- 地域：**华东**等大陆节点（如 `oss-cn-hangzhou`）
- 读写权限：公共读（或通过 CDN 回源私有桶）
- 开启 **静态网站托管**，默认首页 `index.html`，404 填 `index.html`（作品集单页需要）

### 2. 上传文件

将 **`cmq个人网站` 整个目录**（含 `index.html`、`assets/`、`demos/`、`css/`、`js/`）上传到 Bucket **根目录**。

可用脚本（需先安装 [ossutil](https://help.aliyun.com/zh/oss/developer-reference/ossutil/) 并配置 AccessKey）：

```powershell
.\scripts\sync-oss.ps1 -Bucket "你的bucket名称"
```

或在控制台 **文件管理** 里拖拽上传。

### 3. 绑定 CDN + 域名

1. 开通 **CDN**，源站选该 OSS Bucket
2. 备案通过后，CDN 添加加速域名：`www.chenmengqiu-portfolio.xyz`
3. **域名解析**（阿里云 DNS）：
   - 删除指向 Vercel 的 `www` CNAME / `@` A 记录
   - 按 CDN 控制台提示添加 **CNAME** 到 CDN 域名

### 4. HTTPS

在 CDN 申请免费 SSL 证书，开启 HTTPS 强制跳转。

## 更新流程（以后改项目）

1. 改对应项目代码 → 再跑 `.\scripts\build-demos.ps1`
2. `推送GitHub.bat`（若仍用 Vercel 备份）或 `.\scripts\sync-oss.ps1` 上传 OSS
3. CDN 刷新缓存（控制台 **刷新预热**）

## 常见问题

**Q：还要给三个项目各绑域名吗？**  
A：不要。一个域名 + `/demos/子路径` 即可。

**Q：备案期间能用吗？**  
A：可以。继续用 `www.chenmengqiu-portfolio.xyz` → Vercel；推送含 `demos/` 的构建后，Vercel 会优先托管静态 Demo（仍可能偏慢）。

**Q：解析改到阿里云后 Vercel 还能用吗？**  
A：可以留作你自己预览；简历只写国内域名。
